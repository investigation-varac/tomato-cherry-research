# Variedades de Tomate Cherry/Uva

## Formato de Documentación

Cada variedad debe incluir:
- Nombre comercial y casa semillera
- Tipo (cherry, grape, datterino)
- Características agronómicas (vigor, tipo de crecimiento)
- Rendimiento esperado (kg/m²/año)
- Calidad de fruto (°Brix típico, firmeza, vida de anaquel)
- Resistencias/tolerancias (TMV, TYLCV, etc.)
- Requerimientos especiales
